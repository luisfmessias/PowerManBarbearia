# Power Man Barbearia - Landing Page

Uma landing page moderna e responsiva para a barbearia Power Man, desenvolvida com React, TypeScript, Tailwind CSS e Framer Motion.

##  Componentes da Landing Page

- **Header** - Navegação fixa com menu responsivo
- **HeroSection** - Seção inicial com destaque visual
- **ServicesSection** - Apresentação dos serviços
- **GallerySection** - Galeria de imagens dos trabalhos
- **AboutSection** - Informações sobre a barbearia
- **IdeaSection** - Seção de ideias/inspirações
- **LocationSection** - Localização da barbearia
- **ContactSection** - Formulário de contato
- **Footer** - Rodapé com informações

##  Começando

### Instalação

```bash
npm install
```

### Desenvolvimento

```bash
npm run dev
```

A aplicação estará disponível em `http://localhost:5173`

### Build para Produção

```bash
npm run build
```

##  Dependências Principais

- **React 18.3** - Biblioteca UI
- **React Router** - Roteamento
- **Framer Motion** - Animações
- **Tailwind CSS** - Estilos
- **TypeScript** - Tipagem estática
- **Vite** - Build tool moderno

##  Features

- Design moderno e responsivo
- Animações fluidas com Framer Motion
- Navegação suave entre seções
- Menu mobile otimizado
- Componentes reutilizáveis com TypeScript
